# yalbalyk

## Виртуальное окружение

### Windows

```powershell
python -m venv .venv
```

```powershell
.\.venv\Scripts\activate
```

### Linux

```shell
python3 -m venv .venv
```

```shell
source ./.venv/Scripts/activate.sh
```

## Установка библиотек:

```shell
pip install -r requirements.txt
```

## Запуск игры

```shell
python main.py
```

## Как работать над кодом?
